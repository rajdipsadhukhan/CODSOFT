# 🎯 Number Guessing Game (Java Console)

## 📌 Description

This is a simple Number Guessing Game built using Java (console-based).
The system generates a random number, and the player has to guess it within limited attempts.

## 🚀 Features

* Random number generation (1–50)
* Limited attempts (10 guesses)
* Simple console interaction
* Instant feedback (Too High / Too Low / Correct)

## 🛠️ Technologies Used

* Java

## ▶️ How to Run

1. Clone the repository
2. Navigate to the folder:

   ```
   cd numberGame
   ```
3. Compile:

   ```
   javac NumberGame.java
   ```
4. Run:

   ```
   java NumberGame
   ```

## 📌 Sample Output

