# 🏧 ATM Interface

## 📌 Description

A Java-based ATM simulation that allows users to perform basic banking operations such as checking balance, depositing, and withdrawing money.

## ⚙️ Features

* Check account balance
* Deposit money
* Withdraw money
* Input validation for transactions
* Menu-driven interface

## 🛠️ Technologies Used

* Java
* OOP (Object-Oriented Programming)
* Console-based interaction

## ▶️ How to Run

1. Compile the files:
   javac ATM.java BankAccount.java

2. Run the program:
   java ATM

## 🎯 Objective

To simulate real-world ATM functionality using Java and demonstrate class interaction.
