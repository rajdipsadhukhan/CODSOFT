# 📊 Student Grade Calculator (Java Console)

## 📌 Description

This is a simple Student Grade Calculator built using Java (console-based).
It takes marks of multiple subjects, calculates total, average, and assigns a grade.

## 🚀 Features

* Input marks for multiple subjects
* Calculates total marks
* Calculates average percentage
* Displays grade based on performance

## 🛠️ Technologies Used

* Java

## ▶️ How to Run

1. Clone the repository
2. Navigate to the folder:

   ```
   cd gradeCalculator
   ```
3. Compile:

   ```
   javac StudentGrade.java
   ```
4. Run:

   ```
   java StudentGrade
   ```
