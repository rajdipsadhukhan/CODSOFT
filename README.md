# 💻 CODSOFT Internship Projects (Java - Console Based)

## 📌 Overview

This repository contains Java console-based projects developed as part of the CODSOFT Internship.
Each project focuses on basic programming concepts, logic building, and user interaction through the console.

---

## 📂 Projects Included

### 🎯 1. Number Guessing Game

* Guess a randomly generated number
* Limited number of attempts
* Feedback: Too High / Too Low / Correct

📁 Folder: `numberGame`

---

### 📊 2. Student Grade Calculator

* Input marks for multiple subjects
* Calculates total and average
* Assigns grade based on performance

📁 Folder: `gradeCalculator`

---

### 🏧 3. ATM Interface

* Simulates ATM operations
* Deposit, Withdraw, Check Balance
* Console-based interaction

📁 Folder: `ATM_Interface`

---

### 🎓 4. Student Management System (JDBC + MySQL)

* Add, View, Search, Update, Delete student records
* Uses MySQL database connectivity (JDBC)
* Demonstrates CRUD operations with database

📁 Folder: `StudentManagementSystem`

## 🛠️ Technologies Used

* Java (Core Java)
* JDBC (Java Database Connectivity)
* MySQL
* Console (CLI-based programs)

---

## ▶️ How to Run Any Project

1. Clone the repository
2. Navigate to the specific project folder:

   ```
   cd folderName
   ```
3. Compile:

   ```
   javac FileName.java
   ```
4. Run:

   ```
   java FileName
   ```
