# 📊 Grade Calculator GUI

## 📌 Description

A Java GUI application that calculates total marks, average percentage, and assigns grades based on user input.

## ⚙️ Features

* Input marks for multiple subjects
* Automatic total and average calculation
* Grade assignment based on percentage
* User-friendly interface

## 🛠️ Technologies Used

* Java
* Swing (GUI)
* OOP Concepts

## ▶️ How to Run

1. Compile the program:
   javac GradeCalculatorGUI.java

2. Run the program:
   java GradeCalculatorGUI

## 🎯 Objective

To implement a GUI-based grading system using Java and strengthen OOP concepts.
